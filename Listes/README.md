# LISTES
-----------------

## DESCRIPTION
Programme en C pour manier les listes

-----------------
## OPÉRATIONS POSSIBLES
* Affichage de la liste
* Création d'une liste
* Recherche d'un élément dans la liste
* Insérer un élément dans la liste
* Supprimer un élément de la liste
* effacer la liste

-----------------
## Utilisation
Compilateur `gcc`
